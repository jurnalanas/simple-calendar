# simple-calendar

Simple Calendar built with Vanilla JavaScript. No frameworks or libraries.

## Building and running on localhost

First install dependencies:

```sh
yarn
```

To run in hot module reloading mode:

```sh
yarn start
```

To create a production build:

```sh
yarn build-prod
```

Update github page:

```sh
yarn build-prod
```

## Running

```sh
node dist/bundle.js
```
